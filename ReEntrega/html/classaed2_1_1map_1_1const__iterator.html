Descripción detallada

template<class Key, class Meaning, class Compare = std::less<Key>>
class aed2::map< Key, Meaning, Compare >::const_iterator

Parte del módulo que implementa los iteradores que no permiten la modificación de los significados.

Este módulo es análogo a aed2::map::iterator, pero no permite la modificación cuando hay aliasing. Sólo se documentan las funcionalidades nuevas. Para las restantes, ver aed2::map::iterator.
